# NexoCetEx Backend

Backend desarrollado con Node.Js y Express para el proyecto NexoCetEx de la materia implementacion de proyecto.

## Tecnologias aplicadas
Node.js
Express
MySql
Firebase Admin SDK
JWT

## Instalacion
npm install

## Variables de entorno:
Configurar:
DB_HOST
DB_PORT
DB_USER
DB_PASSWORD
DB_NAME
JWT_SECRET
CHAT_SECRET
FIREBASE_SERVICE_ACCOUNT

## Ejecucion
node server.js
